In logics, it is common to associate a formula with the set of assignments that satisfy it.
\[ 0 < \phi < 3 = \{ \phi = 1, \phi = 2 \} \]

In temporal logics, we associate a formula with the set of states in which the formula is true.
\[ [\![ \bm{EX} (b \land c) ]\!]_{M} = \{ s_0, s_3 \} \]

Formal Definition

\[ [\![ f ]\!]_{M} := \{ s \in S \mid M, s \models f \} \]

Set Atomic Proposition

We can think \( \{ s_0, ..., s_n \} \) as an atomic proposition because:
  • \( M, s_0 \models a \)
  • \( M, s_0 \models \{ s0, s_3, ... \} \)
This formal syntax is:

If \( \{ s_0, ..., s_n \} \in S \), then \( \{ s_0, ..., s_n \} \) is a CTL* state formula called set atomic proposition.

Extending CTL* semantics

1a.
\( M, s \models \{ s_0, s_1, ..., s_n \} \)
\( \Leftrightarrow \)
\( s \in \{ s_0, s_1, ..., s_n \} \)
