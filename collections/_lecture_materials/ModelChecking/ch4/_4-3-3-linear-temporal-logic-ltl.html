LTL formulas are of the form \( \bm{A} f \):
  • Only \( \bm{A} \) is allowed.
  • \( f \) is a CTL* path formula that does not include path quantifiers.

Syntax of LTL

LTL formulas:
E0
If \( f \) is an LTL path formula, then \( \bm{A} f \) is a LTL formula.
LTL path formulas:
E1
If \( p \in AP \), then \( p \) is an LTL path formula.
E2
If \( f \) is an LTL path formula, then \( \lnot f, \bm{X} f, \bm{F} f \) and \( \bm{G} f \) are LTL path formulas.
E3
If \( f \) and \( g \) are LTL path formulas, then \( f \land g, f \lor g, f \bm{U} g \) and \( f \bm{R} g \) are LTL path formulas.

Note that \( \bm{X} \) and \( \bm{U} \) are sufficient to represent other operators. (See Some properties)

We can omit \( \bm{A} \) because LTL formulas must have \( \bm{A} \).
  • \( M \models f \) means \( M \models \bm{A} f \).
